# Nền tảng Quản lý Đặt Sân Bóng Online
(Football Field Booking Manager)

Ứng dụng Backend xây dựng trên nền tảng NestJS, TypeScript và MySQL nhằm hỗ trợ khách hàng đặt sân bóng trực tuyến, quản lý lịch đặt sân và theo dõi tình trạng sân theo thời gian thực.

---

# 1. Thực trạng & Ý tưởng dự án

Hiện nay nhiều sân bóng mini vẫn quản lý lịch đặt bằng sổ sách hoặc tin nhắn điện thoại. Việc này dễ dẫn đến:

- Trùng lịch đặt sân
- Khó quản lý khách hàng
- Mất thời gian kiểm tra sân trống
- Khó thống kê doanh thu

**Giải pháp:**

Hệ thống cung cấp các API CRUD tập trung, kết nối trực tiếp tới cơ sở dữ liệu MySQL.
